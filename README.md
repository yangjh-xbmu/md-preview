# md-preview

Quickly preview Markdown files as rendered HTML with GitHub-style styling.

## Quick Start

### Prerequisites

- Go 1.25 or newer

### Install

```bash
go install github.com/yangjh-xbmu/md-preview@latest
```

### Run

```bash
md-preview README.md
```

The command should open a local browser preview of the rendered Markdown file.
